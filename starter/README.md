# Private workspace starter

Copy the contents of this directory into a separate private Git workspace. It creates a minimal operational memory for one user and their agents.

After copying:

1. Replace template text in `map.md` with the user's approved scope.
2. Register the first agent in `operations/agents.md`.
3. Capture the first request in `operations/inbox.md`.
4. Keep `operations/now.md` limited to three current priorities.
5. Write one daily report per active agent.

Do not copy a user's actual records back into the public repository.
